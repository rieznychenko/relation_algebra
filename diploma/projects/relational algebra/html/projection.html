Проекция (PROJECTION)

При выполнении проекциии ортношения на заданный набор атрибутов производится отношение, кортежи которого создаются путем взятия соответствующих значений из кортежей отношений.

Проекцией отношения R по атрибутам A, где каждый из атрибутов принадлежит отношению R, называется отношение с заголовком A и телом, содержащим множество кортежей вида a, таких, для которых в отношении R найдутся кортежи со значением атрибута A равным a.

Синтаксис операции проекции:R[A].

Операция проекции дает "вертикальный срез" отношения, в котором удалены все возникшие при таком срезе дубликаты кортежей.
